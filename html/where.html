How to get there

If you want to travel to this fantastic world, there are three main ways

Get a book!!

This is by far the best way, reading, lets you explore this fantastic world, your mind will build upon the descriptions provided by Rowling's. If you need help getting them, I would recommend going to a library. These are unique buildings where there are tons of books, its an awesome place where you can read them!! See the movies

Seeing the movies is also a great way to enjoy this fantastic world, I can assure you, it will be some well spent time. If you wanna watch them all a streaming service who-must-not-be-named comes to mind or you can simply get the DVDs

The third way

There is a third way, if happens to receive an invitation taken to you by an owl (or a bunch of them) telling you that you have been accepted in Hogwarts, well congrats! You are in for a treat!
